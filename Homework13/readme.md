# Міні-гра "словко".
Для роботи гри потрібен файл зі словами, які можуть бути обрані для вгадування.
Файл повинен мати назву library.txt, але може мати й довільну назву, тоді програма запропонує ввести шлях до файлу.
Гра має 2 режими складності: простий (по замовчуванню) та складний.
У простому режимі усі букви які були введені і знайдені в слові - залишаються виведеними у наступних спробах.
У складному режимі усі букви заново закриваються "зірочками" у наступних спробах відгадати слово.
Щоб змінити режим, запустити гру повторно чи вибрати "слово дня" - не потрібно перезапускати програму, це все можна зробити з меню.

## Формат
Формат файлу: word:bool.
word - слово для відгадування.
bool-значення - слово відгадане як "слово дня".
Якщо у фафлі будуть помилки - програма це зафіксує і виведе помилку.
Якщо у файлі будуть пусті стрічки - проблема це теж зафіксує і проігнорує.

## Слово дня
Система обирає по номеру дня місяця слово зі списику, тому в файлі зі словами треба мати слів більше як 31.
Система зберігає в файл статус "виконано", якщо "слово дня" було відгадане.

## Issues
